== Description ==

With the Auth Armor plugin, you can login using your phone without passwords! 

More secure, faster and best of all, nothing to remember or type in.

== Installation ==

Upload the auth-armor folder to the /wp-content/plugins/ directory

Activate the plugin through the ‘Plugins’ menu in WordPress.

Enable and configure the widget as usual on the Design / Widgets page.

== Frequently Asked Questions ==

= Is logging in without a password safe? =

Yes! It's more secure. Why? Because with passwords, a hacker could steal a database. Or you could lose your password. With Auth Armor, only your phone and biometrics can log in.

= Do I need the Auth Armor app to login? =

Yes, logging in with this plugin requires the Auth Armor Authenticator app. Please see instructions during plugin install.

= Can I use a security key? =

Yes, Auth Armor supports using any FIDO enabled security key.
